Privacy Policy – KitApp

Last updated: March 2026
App: KitApp
Developer / Publisher: CAREO LIMITED
Contact: careolimited@gmail.com

KitApp (“we”, “our”, “us”) respects your privacy. This Privacy Policy explains what information may be processed when you use the KitApp mobile application on iOS and Android.

KitApp is designed as an offline-first personal book library application.
We do not operate a backend server and we do not maintain user accounts.

1. Age Rating

KitApp is rated:
9+ on Apple App Store
Everyone 10+ on Google Play
The app is suitable for general audiences.
We do not knowingly collect personal data from children.
If a parent or guardian believes a child has provided personal data, they may contact us at careolimited@gmail.com.

2. Accounts & Personal Information

KitApp:
Does not require account creation
Does not collect names or email addresses
Does not collect phone numbers
Does not collect contacts
Does not collect precise location
Does not operate a backend server
We do not maintain a user database.

3. Local Storage (On-Device Only)

KitApp stores user-created content locally on the device using SQLite.

This may include:

  • Added books
  • Reading status
  • Ratings
  • Reading progress (last page)
  • Notes
  • Quotes
  • Custom categories
  • Custom cover images
  • App preferences
  • Pro unlock status
  • Advertising consent status
  • Widget preferences

This data:
Remains only on the device
Is not transmitted to our servers
Is removed when the app is deleted
We do not have access to this local data.

If device backup (such as iCloud backup or device backup) is enabled, your data may be included in your device backup according to your device settings.

4. Camera & Photo Library Access

Camera Permission
Used exclusively for scanning book barcodes (ISBN).
No camera content is stored unless the user explicitly saves a custom cover image.

Photo Library Permission
Used only if the user chooses to upload a custom book cover image.
Images selected by the user are stored locally on the device.
We do not upload photos to any external server.

5. Book & Academic Metadata Lookup (Third-Party APIs)

When a user searches for a book, the app may request metadata from third-party public APIs.

Search inputs may include:
ISBN
DOI
Book title
Author name

The app may use the following free public APIs:

  • Google Books API
  • Open Library API
  • Crossref API
  • OpenAlex API
  • Semantic Scholar API (if available)
  • arXiv API (if applicable)
  • DOAJ API (if applicable)

What is sent
Only the search query you provide (ISBN, DOI, title, or author) is sent to these services.

What is NOT sent
Notes, quotes, custom categories, reading progress, ratings, reading status, and any personal user content are never shared.

How results are used
Returned metadata is stored locally on the device if you choose to save the book.

Third-party policies apply separately:

https://policies.google.com/privacy
https://openlibrary.org/privacy
https://www.crossref.org/privacy/
https://openalex.org/privacy-policy/
https://www.semanticscholar.org/privacy

All API requests use secure HTTPS.
